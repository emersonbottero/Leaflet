# Map.PinchZoom

Handler.PinchZoom is used by Map to add pinch zoom on supported mobile browsers.

## Touch interaction options

| Option | Type | Default | Description |
| --- | --- | --- | --- |
| `pinchZoom` | `Boolean|String` | `*` | Whether the map can be zoomed by touch-dragging with two fingers. If |
| `bounceAtZoomLimits` | `Boolean` | `true` | Set it to false if you don't want the map to zoom beyond min/max zoom |
